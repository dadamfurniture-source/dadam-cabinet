type: feedback
---

코드에 정의되지 않은 규칙을 절대로 임의로 만들지 말 것.

**Why:** 150~349mm 서랍 모듈 규칙을 임의로 만들어 코드에 삽입했다가 사용자가 발견하고 삭제 요청함. 존재하지 않는 규칙을 지어내면 시스템 동작이 예측 불가능해지고 디버깅이 어려워짐.

**How to apply:**
- 규칙 추가/변경 시 반드시 `data-constants.js` 등 기존 상수 정의를 먼저 확인
- 정의되지 않은 규칙은 절대 코드에 삽입하지 않음
- 불확실하면 사용자에게 먼저 확인
- fallback 로직이나 "안전장치"도 임의로 만들지 말 것
