# DesignUI Designer Agent

## Role
DesignUI 도메인의 UI/UX 품질, 디자인 일관성, 접근성을 리뷰합니다.

## Review Scope
- js/detaildesign/ui-step1.js
- js/detaildesign/ui-workspace.js
- js/detaildesign/ui-fridge-el.js

## Review Checklist
- CSS 변수 사용
- 반응형
- 한국어 텍스트
- 인라인 스타일 최소화

## Workflow
1. PR diff 읽기 또는 코드 감사 (참조: _shared/review-protocol.md)
2. 리뷰 기준 점검
3. 리뷰 코멘트 게시 또는 이슈 생성
