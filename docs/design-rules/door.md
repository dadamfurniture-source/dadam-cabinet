# 도어교체 (Door) / 비규격장 (Custom) 설계 규칙

> categoryId: `door` / `custom`

## 1. 도어교체 (door)

| 항목 | 값 |
|------|-----|
| 기본 깊이 | 18mm (도어 두께) |
| 워크스페이스 | 범용 (sink-style) |

도어교체는 기존 가구의 도어만 교체하는 특수 카테고리.
- 도어 치수(W x H) 입력
- 도어 재질/색상/마감 선택
- BOM: 도어 자재만 추출

## 2. 비규격장 (custom)

| 항목 | 값 |
|------|-----|
| 기본 깊이 | 0mm (사용자 입력) |
| 워크스페이스 | 범용 (sink-style) |

비규격장은 정해진 규격이 없는 자유 설계 카테고리.
- 모든 치수를 사용자가 직접 입력
- 상부/하부 모듈 자유 구성

## 3. EL 도어 타입 (공통 참조)

| ID | 이름 | 설명 |
|----|------|------|
| swing | 여닫이 | 기본 도어 타입 |
| slide | 슬라이딩 | 좌우 이동 |
| lift | 리프트업 | 위로 열림 |

|----|------|
