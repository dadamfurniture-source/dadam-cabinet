## 1. 도어 분배 규칙 (v28)

### 1.1 상수
| 상수 | 값 | 설명 |
|------|-----|------|
| DOOR_TARGET_WIDTH | 450mm | 목표 도어 너비 |
| DOOR_MAX_WIDTH | 600mm | 최대 도어 너비 |
| DOOR_MIN_WIDTH | 350mm | 최소 도어 너비 |
| MIN_REMAINDER | 5mm | 최소 잔여 (시공 공차) |
| MAX_REMAINDER | 10mm | 최대 잔여 허용 |

### 1.2 도어 너비 선택 우선순위
```
1순위: 도어 수 최소 (넓은 모듈 우선)
2순위: 목표 450mm에 가까운 너비
3순위: 잔여 공간이 작은 것
```

### 1.3 잔여 공간 처리
| 마감 타입 | 우선 잔여 | 차선 잔여 |
|-----------|----------|----------|
| 비몰딩 (Filler/EP/None) | 최대 10mm 허용 |
| 몰딩 (Molding) | 최대 10mm 허용 |
- 잔여 ≤10mm: 시공 공차로 허용 (모듈에 흡수 안 함)

### 1.4 모듈 패턴 (2D/1D)
```
doorCount / 2 = 몫(2D 모듈) + 나머지(1D 모듈)
2D 모듈 너비 = doorWidth × 2
1D 모듈 너비 = doorWidth
```

### 1.5 상부장 대칭 배치
```
도어 3개 이상: 1D(좌 가장자리) + 2D(중앙) + 1D(우 가장자리)
예: 
1. 5도어 → 1D + 2D + 1D + 1D OR 1D + 2D + 2D
2. 6도어 → 1D + 2D + 2D + 1D
```

### 1.6 하부장 표준 배치
```
2D 먼저 + 1D 나중 (대칭 없음)
예: 5도어 → 2D + 2D + 1D
```

### 1.7 Dead Zone (601~689mm)
- 갭이 601~689mm → 도어 1개에는 너무 크고 2개에는 너무 작음
- 해결: 서랍형으로 자동 전환

### 1.8 소규격 갭 (<350mm) 흡수
- 350mm 미만 갭은 모듈 생성 불가
- 인접한 큰 갭에 균등 분배하여 흡수

## 2. 유효 공간 계산

### 2.1 공식
```
유효공간 = 전체W - 좌측마감 - 우측마감 - 코너1마감(L/U) - 코너2마감(U)
```

### 2.2 레이아웃별 코너 적용
| 레이아웃 | 코너1 | 코너2 |
|---------|-------|-------|
| ㅡ자형 (I) | 없음 | 없음 |
| ㄱ자형 (L) | 적용 | 없음 |
| ㄷ자형 (U) | 적용 | 적용 |

## 10. 냉장고장 규칙

### 10.1 높이 상수
| 항목 | 값 |
|------|-----|
| 상부장 최대 높이 | 400mm |
| 좌대 높이 | 60mm |
| 냉장고 상단 간격 | 15mm (고정) |
| 상몰딩 기본 | 50mm |
| 모듈 기본 깊이 | 550mm |
| 설치 필요 깊이 | 700mm |
| EL장/키큰장 기본 폭 | 600mm |

### 10.2 높이 자동 계산 (55:45 비율)
```
상부장H = min(400, 전체H - 냉장고H - 15 - 상몰딩H)
모듈바디H = 전체H - 상몰딩H - 상부장H - 좌대H
중단H = floor(모듈바디H × 0.55)
하단H = floor(모듈바디H - 중단H)
```

### 10.3 브랜드별 여유 간격
| 브랜드/타입 | 좌우 간격 | 유닛간 간격 |
|-----------|----------|-----------|
| LG 빌트인 | 4mm | 8mm |
| LG 프리스탠딩 | 50mm | 0mm |
| 삼성 비스포크 | 12mm | 10mm |
| 삼성 인피니트 | 5mm | 10mm |
| 삼성 프리스탠딩 | 50mm | 0mm |

### 10.4 냉장고장 너비 계산
```
냉장고W = 좌우간격×2 + 유닛합산W + (유닛수-1)×유닛간간격
```
